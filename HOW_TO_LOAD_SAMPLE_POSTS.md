1. CJ CGV (079160)
2. 헬릭스미스 (084990)
3. 카티스 (035720)

## 방법 1: 브라우저 콘솔에서 실행 (권장)

1. `index.html` 파일을 브라우저에서 엽니다
2. F12 키를 눌러 개발자 도구를 엽니다
3. Console 탭을 클릭합니다
4. 아래 코드를 복사하여 콘솔에 붙여넣고 Enter를 누릅니다:

```javascript
// sample-posts.js 파일의 전체 내용을 복사하여 실행
```

또는 `sample-posts.js` 파일을 직접 열어서 전체 내용을 복사한 후 콘솔에 붙여넣습니다.

## 방법 2: index.html에 스크립트 추가

1. `index.html` 파일을 엽니다
2. 마지막 줄 근처의 주석을 찾습니다:
   ```html
   <!-- <script src="sample-posts.js"></script> -->
   ```
3. 주석을 제거합니다:
   ```html
   <script src="sample-posts.js"></script>
   ```
4. `index.html`을 저장하고 브라우저에서 새로고침합니다
5. 샘플 포스트가 로드되면 다시 주석 처리합니다 (중복 방지)

## 확인 방법

샘플 포스트가 로드되면:
- 브라우저를 새로고침(F5)합니다
- 홈페이지에 3개의 종목 분석 글이 표시됩니다
- 각 글을 클릭하여 상세 내용을 확인할 수 있습니다

## 주의사항

- 샘플 포스트는 localStorage에 저장됩니다
- 브라우저 데이터를 삭제하면 샘플 포스트도 삭제됩니다
- 같은 스크립트를 여러 번 실행하면 중복으로 저장될 수 있습니다
