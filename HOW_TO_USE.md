# 사용 방법

## 즉시 실행하기

1. **`index.html` 파일을 더블클릭**
   - 브라우저가 자동으로 열립니다
   - 바로 블로그를 사용할 수 있습니다!

2. **인터넷 연결 확인**
   - CDN에서 라이브러리를 불러오므로 인터넷 연결이 필요합니다
   - 첫 로딩 후에는 캐시되어 더 빠르게 로드됩니다

## 기능 사용하기

### 글 작성하기
1. 상단 메뉴에서 "글 작성" 클릭
2. 제목, 본문, 태그, 이미지 등을 입력
3. "글 발행하기" 버튼 클릭
4. 작성한 글이 자동으로 저장됩니다

### 글 보기
1. 홈페이지에서 글 카드를 클릭
2. 상세 페이지에서 전체 내용 확인
3. PC에서는 오른쪽에 최신글 목록 표시
4. 모바일에서는 글 하단에 최신글 목록 표시

### 데이터 관리
- 모든 데이터는 브라우저의 localStorage에 저장됩니다
- 브라우저 개발자 도구(F12)에서 확인 가능
- 데이터 백업: 개발자 도구 > Application > Local Storage에서 복사

## 문제 해결

### 페이지가 제대로 표시되지 않을 때
- 인터넷 연결을 확인하세요
- 브라우저를 새로고침하세요 (F5)
- 브라우저 캐시를 삭제하세요

### 글이 저장되지 않을 때
- 브라우저가 localStorage를 지원하는지 확인
- 시크릿 모드에서는 데이터가 저장되지 않을 수 있습니다
- 브라우저 설정에서 쿠키/localStorage가 차단되어 있지 않은지 확인

### 스타일이 깨질 때
- Tailwind CSS CDN이 제대로 로드되었는지 확인
- 브라우저 콘솔(F12)에서 오류 메시지 확인
