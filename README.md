# 주식 종목 분석 블로그

주식 종목 분석과 주가 전망을 제공하는 전문 블로그입니다.

## 🎉 Node.js 불필요!

이 블로그는 **순수 HTML/CSS/JavaScript**로 만들어져 있어서:
- ✅ Node.js 설치 불필요
- ✅ 빌드 과정 불필요
- ✅ 웹 서버 없이도 실행 가능
- ✅ `index.html` 파일을 더블클릭하면 바로 실행!

## 실행 방법

### 방법 1: 직접 실행 (가장 간단)
1. `index.html` 파일을 더블클릭
2. 브라우저에서 자동으로 열림
3. 바로 사용 가능!

### 방법 2: 웹 서버에 업로드
1. `index.html`, `app.js`, `styles.css` 파일을 웹 서버에 업로드
2. 웹 브라우저로 접속
3. 완료!

## 주요 기능

- 📝 블로그 글 작성 및 이미지 첨부
- 📱 반응형 디자인 (PC/모바일)
- 🔍 SEO 최적화 (구글 검색 엔진 최적화)
- 🏷️ 태그 시스템
- 📊 종목 정보 연동
- 💾 클라이언트 사이드 데이터 저장 (localStorage)
- ⚡ Node.js 없이 즉시 실행

## 기술 스택

- 순수 HTML/CSS/JavaScript
- React (CDN)
- Tailwind CSS (CDN)
- Marked.js (Markdown 파서, CDN)
- localStorage (데이터 저장)

## 사용 방법

1. **글 작성**: `#/write` 페이지에서 새 글을 작성할 수 있습니다.
2. **이미지 첨부**: 이미지 URL을 입력하여 이미지를 첨부할 수 있습니다.
3. **태그 추가**: SEO를 위해 태그를 추가하세요.
4. **종목 정보**: 종목명과 종목코드를 입력할 수 있습니다.

## 데이터 저장

- 블로그 포스트는 **브라우저의 localStorage**에 저장됩니다
- 각 사용자의 브라우저에 독립적으로 저장됩니다
- 브라우저 데이터를 삭제하면 포스트도 삭제됩니다

## SEO 최적화 기능

- 각 게시글의 제목이 웹문서 타이틀이 됩니다
- 태그가 메타 태그 description에 포함됩니다
- 구조화된 데이터 (JSON-LD) 지원
- Open Graph 및 Twitter Card 지원
- 반응형 디자인

## 레이아웃

- **PC**: 최신글은 오른쪽 사이드바에 표시됩니다
- **모바일**: 최신글은 글 하단에 표시됩니다

## 파일 구조

```
stock_blog/
├── index.html      # 메인 HTML 파일 (이 파일을 열면 실행!)
├── app.js          # React 앱 코드
└── styles.css      # 커스텀 스타일
```

## 주의사항

- 데이터는 브라우저의 localStorage에 저장되므로, 다른 브라우저나 기기에서는 데이터가 공유되지 않습니다
- 인터넷 연결이 필요합니다 (CDN 라이브러리 로드를 위해)
- 오프라인 사용을 원하면 CDN 대신 로컬 파일을 사용하도록 수정해야 합니다

## 배포

웹 서버에 배포하려면:
1. `index.html`, `app.js`, `styles.css` 파일을 업로드
2. 완료!

GitHub Pages, Netlify, Vercel 등 어떤 정적 호스팅 서비스에도 배포 가능합니다.
